# AGENTS.md

## 目的

このリポジトリは、ブラウザから PDF を編集し、新しい PDF としてダウンロードできるアプリケーションを実装するためのものです。

対象機能:

- ページの回転
- ページの分割
- ページの並べ替え
- ページの切り出し
- 編集結果を新しい PDF としてダウンロード

## 前提

- サーバー環境は Ubuntu を想定する
- ローカル開発と本番配備の差分を減らすため、Docker での起動を優先する
- フロントエンドとサーバー処理は同一アプリケーション内にまとめる

## 採用方針

- Web アプリケーションは TypeScript を使って実装する
- UI はブラウザで完結させ、PDF の最終加工はサーバー側で実施する
- PDF 編集ロジックは純粋関数寄りに分離し、API 層から直接複雑な処理を書かない
- 一時ファイルに依存しすぎず、可能な範囲でメモリ上で処理を完結させる

## 実装ルール

- 日本語で読める命名や文書を優先する
- コメントは必要最小限に留め、処理の意図が伝わりにくい箇所だけに付ける
- 画面の状態と PDF 編集コマンド定義は分離する
- サーバー API は入力バリデーションを持たせる
- 失敗時はユーザーが再操作しやすいエラーメッセージを返す

## 推奨ディレクトリ構成

- `src/app`: 画面と API 入口
- `src/components`: UI コンポーネント
- `src/lib`: PDF 編集処理、型、共通処理
- `public`: 静的ファイル

## 動作確認

- 変更後は少なくとも型チェックまたはビルドを実行する
- PDF 編集処理を変更した場合は、代表的な操作パターンを想定した検証を追加する

## コミット方針

- コミットメッセージは日本語で記述する
- 大きすぎるコミットは避け、機能単位または責務単位で分割する
- 自動生成物を含める場合は、その理由が説明できる状態にする
